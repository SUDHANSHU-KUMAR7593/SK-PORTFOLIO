Case Study

AI Resume Analyzer

A Flask-based ATS resume analyzer that compares an uploaded resume against a target job description and returns a practical fit score, keyword gaps, structure checks, and improvement recommendations.

Problem

Students and job seekers often submit the same resume everywhere, without knowing whether it contains the right keywords, standard sections, contact details, or measurable impact for a specific role.

Solution

I built a web app where users paste a job description, upload a PDF, DOCX, or TXT resume, and receive an analysis report with a 0-100 score, verdict, matched keywords, missing keywords, and prioritized fixes.

Backend

  • Flask API with CORS support and a health check route.
  • Secure file upload handling with UUID-based filenames.
  • PDF, DOCX, and TXT text extraction using PyPDF2 and python-docx.
  • Upload validation for file type, empty files, readable text, and 8 MB size limit.

Scoring Logic

  • 45% weighted job keyword alignment.
  • 20% resume structure detection for sections like Summary, Experience, Education, Skills, Projects, and Certifications.
  • 15% impact evidence using bullets, metrics, and action verbs.
  • 10% contact readiness and 10% readability.
01 Upload

User uploads PDF, DOCX, or TXT resume and adds a target job description.

02 Extract

Backend extracts readable text and rejects unsupported or low-text files.

03 Analyze

Algorithm compares keywords, sections, evidence, contact info, and readability.

04 Improve

Frontend displays score breakdown, missing keywords, and clear next steps.

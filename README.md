# Sudhanshu Kumar Portfolio

A responsive personal portfolio for Sudhanshu Kumar, built with HTML, CSS, and
JavaScript. It highlights full-stack development, Python backend work,
AI-focused projects, internships, certifications, resume access, and contact
links.

## Files

- `index.html` - portfolio content and sections
- `style.css` - responsive visual design and theme styles
- `script.js` - theme toggle, mobile navigation, scroll states, animations, and form UX
- `profile.png` - profile image
- `MYRESUME.pdf` - resume download/view file

Open `index.html` in a browser to view the site.
